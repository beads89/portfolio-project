Branching Test

In this project, a fellow student and I learned about the wonders of branching. We followed the following steps to practice this concept.

  1. Create and code a webpage on the "main" branch.
  2. Create "red" and "blue" theme branches to style with.
  3. Merge our chosen style to "main" and make sure commits and changes merged correctly.
  4. Delete "red" and "blue" branches to present finished product without unnecessary branches on Github.

Languages used:

  • HTML
  • CSS
